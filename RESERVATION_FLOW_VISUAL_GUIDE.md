## 🎨 Color Coding & Themes

### Step 1: Legal Gate Advisory
- **Primary Color:** Amber (#FCD34D)
- **Badge:** 72H Security Lock (amber-50 bg, amber-300 border)
- **Cards:** Blue, Green, Purple (compliance requirements)
- **Button:** Gold background (#D4AF37)

### Step 2: Reservation Attribution
- **Self-Led:** Gold highlight (fcGold)
- **Agent Assisted:** Slate/gray highlight (fcSlate)
- **Referral:** Blue highlight (#2563EB)
- **Button:** Slate background

### Step 3: KYC Verification
- **Primary Color:** Purple (#9333EA)
- **Badge:** Required Documentation (purple-50 bg, purple-300 border)
- **Upload Zone:** Dashed border, purple accent
- **Button:** Purple background

### Step 4: Legal Acceptance
- **Primary Color:** Red (#DC2626)
- **Badge:** Binding Agreement (red-50 bg, red-300 border)
- **Document:** Scrollable with gray background
- **Button:** Red background (Execute)

### Step 5: Success Confirmation
- **Primary Color:** Green (#16A34A)
- **Icon:** Large green checkmark
- **Badge:** Amber countdown timer
- **Button:** Slate background (Enter Dashboard)

## 📱 Responsive Behavior

### Mobile (< 768px)
- Modal width: 90vw (mx-4)
- Font sizes: Slightly smaller (text-2xl → text-xl for headers)
- Button padding: py-4 (tall tap targets)
- Progress stepper: Horizontal scroll if needed

### Tablet (768px - 1024px)
- Modal width: Max-width 2xl (672px)
- Standard font sizes
- Full visibility of all elements

### Desktop (> 1024px)
- Modal width: Max-width 2xl (672px)
- Centered with backdrop blur
- Mouse hover states active
- Smooth transitions

## ⚡ Validation Rules

### Step 1 → Step 2
- ✅ `advisoryAccepted === true`
- ❌ Shows alert if not checked

### Step 2 → Step 3
- ✅ `attributionType === 'self'` (always valid)
- ✅ `attributionType === 'agent' && selectedAgent !== null`
- ✅ `attributionType === 'referral' && referralCode !== ''`
- ❌ Shows alert if agent/referral incomplete

### Step 3 → Step 4
- ✅ `fullName !== ''`
- ✅ `idNumber !== ''`
- ✅ `contactPhone !== ''`
- ✅ `idDocumentUrl !== ''` (file uploaded)
- ❌ Shows alert if any field empty

### Step 4 → Step 5
- ✅ `agreementAccepted === true`
- ✅ `termsAccepted === true`
- ❌ Shows alert if either checkbox unchecked

### Step 5 → Dashboard
- Always valid - navigates to `/dashboards/client`

## 🧪 Testing Scenarios

### Happy Path
1. User clicks stand on map
2. Modal opens to Step 1 (Advisory)
3. User checks acknowledgment → Next
4. User selects "Self-Led" → Next
5. User fills all KYC fields → Uploads ID → Next
6. User checks both legal agreements → Execute
7. Success screen shows → Digital ref generated
8. User clicks "Enter Dashboard" → Redirects

### Edge Cases
- Empty agent list → "Self-Led" still works
- Invalid file upload → Error toast, retry
- Network error during execution → Error alert, retry
- Mobile landscape → Modal adapts height
- Slow connection → Loading states visible
