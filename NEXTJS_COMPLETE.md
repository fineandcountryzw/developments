## 📞 Quick Help

**"How do I..."**

- Start development server? → `npm run dev`
- Test an API route? → `curl http://localhost:3000/api/admin/developments`
- Open database? → `npx prisma studio`
- Build for production? → `npm run build`
- Fix a broken build? → `rm -rf .next && npm run build`
- Check for errors? → `npm run type-check`

**"Something is broken"**

1. Check terminal for errors
2. Check browser console (F12)
3. Check NEXTJS_TROUBLESHOOTING.md
4. Try: `rm -rf .next && npm run dev`
